# <a name="configure-the-chargeable-components-of-a-project-based-quote-line"></a><span data-ttu-id="0ff68-103">Configurar os componentes faturáveis de uma linha de proposta baseada em projetos</span><span class="sxs-lookup"><span data-stu-id="0ff68-103">Configure the chargeable components of a project-based quote line</span></span>

<span data-ttu-id="0ff68-104">_**Aplica-se A:** Project Operations para cenários baseados em recursos/não armazenados_</span><span class="sxs-lookup"><span data-stu-id="0ff68-104">_**Applies To:** Project Operations for resource/non-stocked based scenarios_</span></span>

<span data-ttu-id="0ff68-105">Uma linha de proposta baseada em projetos pode ter incluído componentes e componentes faturáveis.</span><span class="sxs-lookup"><span data-stu-id="0ff68-105">A project-based quote line can have included components and chargeable components.</span></span>

<span data-ttu-id="0ff68-106">Os componentes incluídos estão sujeitos ao método de faturação, às divisões de clientes, aos limites a não exceder e às definições de frequência de faturação definidas na linha de proposta baseada em projetos.</span><span class="sxs-lookup"><span data-stu-id="0ff68-106">Included components are subject to the billing method, customer splits, not-to-exceed limits, and invoice frequency settings defined on the project-based quote line.</span></span>

   - <span data-ttu-id="0ff68-109">Faturável</span><span class="sxs-lookup"><span data-stu-id="0ff68-109">Chargeable</span></span>
   - <span data-ttu-id="0ff68-110">Não faturável</span><span class="sxs-lookup"><span data-stu-id="0ff68-110">Non-chargeable</span></span>

<span data-ttu-id="0ff68-111">Os componentes faturáveis podem ser definidos nas categorias de funções e transações.</span><span class="sxs-lookup"><span data-stu-id="0ff68-111">Chargeable components can be defined on roles and transaction categories.</span></span>

<span data-ttu-id="0ff68-112">A capacidade de cobrança definida em funções para uma linha de proposta de projeto aplica-se apenas à classe de transação **Tempo**.</span><span class="sxs-lookup"><span data-stu-id="0ff68-112">Chargeability that is defined on roles for a project quote line only applies to the **Time** transaction class.</span></span> <span data-ttu-id="0ff68-113">Se uma linha de proposta do projeto tiver **Incluir Tempo** = **Não**, o separador **Funções Faturáveis** não está disponível.</span><span class="sxs-lookup"><span data-stu-id="0ff68-113">If a project quote line has **Include Time** = **No**, the **Chargeable Roles** tab isn't available.</span></span>

<span data-ttu-id="0ff68-114">A capacidade de cobrança definida em categorias de transação para uma linha de proposta de projeto aplica-se apenas à classe de transação **Despesa**.</span><span class="sxs-lookup"><span data-stu-id="0ff68-114">Chargeability that is defined on transaction categories for a project quote line only applies to the **Expense** transaction class.</span></span> <span data-ttu-id="0ff68-115">Se uma linha de proposta do projeto tiver **Incluir Despesas** = **Não**, o separador **Categorias Faturáveis** não está disponível.</span><span class="sxs-lookup"><span data-stu-id="0ff68-115">If a project quote line has **Include Expenses** = **No**, the **Chargeable Categories** tab isn't available.</span></span>
