<span data-ttu-id="3fe65-105">Uma linha de proposta baseada em projeto tem o conceito de componentes *incluídos* e componentes *faturáveis*.</span><span class="sxs-lookup"><span data-stu-id="3fe65-105">A project-based quote line has the concept of *included* components and *chargeable* components.</span></span>

<span data-ttu-id="3fe65-106">Os componentes incluídos estão sujeitos a:</span><span class="sxs-lookup"><span data-stu-id="3fe65-106">Included components are subject to:</span></span>

  - <span data-ttu-id="3fe65-109">Definições de frequência de fatura definidas na linha de proposta baseada em projeto</span><span class="sxs-lookup"><span data-stu-id="3fe65-109">Invoice frequency settings defined on the project-based quote line</span></span>

<span data-ttu-id="3fe65-110">Um subconjunto dos componentes incluídos pode ser marcado como faturável utilizando o campo **Tipo de Faturação**.</span><span class="sxs-lookup"><span data-stu-id="3fe65-110">A subset of the included components can be marked as chargeable using the **Billing Type** field.</span></span> <span data-ttu-id="3fe65-111">O campo **Tipo de Faturação** é um conjunto de opções que permite os seguintes valores no contexto de uma linha de proposta:</span><span class="sxs-lookup"><span data-stu-id="3fe65-111">The **Billing Type** field is an option-set that allows the following values in the context of a quote line:</span></span>

  - <span data-ttu-id="3fe65-113">Não faturável</span><span class="sxs-lookup"><span data-stu-id="3fe65-113">Non-chargeable</span></span>

<span data-ttu-id="3fe65-114">Os componentes faturáveis podem ser definidos nas categorias de tarefas, funções e transações.</span><span class="sxs-lookup"><span data-stu-id="3fe65-114">Chargeable components can be defined on tasks, roles, and transaction categories.</span></span>

### <a name="update-a-project-task-to-be-chargeable-or-non-chargeable"></a><span data-ttu-id="3fe65-121">Atualizar uma tarefa de projeto para faturável ou não faturável</span><span class="sxs-lookup"><span data-stu-id="3fe65-121">Update a project task to be chargeable or non-chargeable</span></span>

<span data-ttu-id="3fe65-122">Uma tarefa do projeto pode ser faturável ou não faturável no contexto de uma linha de proposta baseada em projeto específica que torna possível a seguinte configuração:</span><span class="sxs-lookup"><span data-stu-id="3fe65-122">A project task can be chargeable or non-chargeable in the context of a specific project-based quote line, which makes the following setup possible:</span></span>
