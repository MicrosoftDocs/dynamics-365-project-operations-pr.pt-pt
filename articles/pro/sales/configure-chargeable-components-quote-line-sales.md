<span data-ttu-id="d7c89-105">Uma linha de proposta baseada em projeto tem o conceito de componentes *incluídos* e componentes *faturáveis*.</span><span class="sxs-lookup"><span data-stu-id="d7c89-105">A project-based quote line has the concept of *included* components and *chargeable* components.</span></span>

<span data-ttu-id="d7c89-106">Os componentes incluídos estão sujeitos a:</span><span class="sxs-lookup"><span data-stu-id="d7c89-106">Included components are subject to:</span></span>

  - <span data-ttu-id="d7c89-107">Método de faturação e divisões de clientes</span><span class="sxs-lookup"><span data-stu-id="d7c89-107">Billing method and customer splits</span></span>
  - <span data-ttu-id="d7c89-108">Limites a não exceder</span><span class="sxs-lookup"><span data-stu-id="d7c89-108">Not-to-exceed limits</span></span> 
  - <span data-ttu-id="d7c89-109">Definições de frequência de fatura definidas na linha de proposta baseada em projeto</span><span class="sxs-lookup"><span data-stu-id="d7c89-109">Invoice frequency settings defined on the project-based quote line</span></span>

  - <span data-ttu-id="d7c89-113">Não faturável</span><span class="sxs-lookup"><span data-stu-id="d7c89-113">Non-chargeable</span></span>

<span data-ttu-id="d7c89-114">Os componentes faturáveis podem ser definidos nas categorias de tarefas, funções e transações.</span><span class="sxs-lookup"><span data-stu-id="d7c89-114">Chargeable components can be defined on tasks, roles, and transaction categories.</span></span>

<span data-ttu-id="d7c89-119">A capacidade de faturação é definida nas categorias de transação de uma linha de proposta e aplica-se apenas à classe de transação **Despesa**.</span><span class="sxs-lookup"><span data-stu-id="d7c89-119">Chargeability is defined on transaction categories for a  quote line and only applies to the **Expense** transaction class.</span></span> <span data-ttu-id="d7c89-120">Se o campo **Incluir Despesas** estiver definido para **Não** numa linha de proposta, o separador **Categorias Faturáveis** não está disponível.</span><span class="sxs-lookup"><span data-stu-id="d7c89-120">If the field, **Include Expenses** is set to **No** on a project quote line, the **Chargeable Categories** tab isn't available.</span></span>

### <a name="update-a-project-task-to-be-chargeable-or-non-chargeable"></a><span data-ttu-id="d7c89-121">Atualizar uma tarefa de projeto para faturável ou não faturável</span><span class="sxs-lookup"><span data-stu-id="d7c89-121">Update a project task to be chargeable or non-chargeable</span></span>

<span data-ttu-id="d7c89-122">Uma tarefa do projeto pode ser faturável ou não faturável no contexto de uma linha de proposta baseada em projeto específica que torna possível a seguinte configuração:</span><span class="sxs-lookup"><span data-stu-id="d7c89-122">A project task can be chargeable or non-chargeable in the context of a specific project-based quote line, which makes the following setup possible:</span></span>
