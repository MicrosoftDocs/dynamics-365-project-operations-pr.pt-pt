# <a name="configuring-chargeable-components-of-a-project-based-contract-line"></a><span data-ttu-id="f55d1-103">Configurar componentes faturáveis de um item de contrato baseado em projeto</span><span class="sxs-lookup"><span data-stu-id="f55d1-103">Configuring chargeable components of a project-based contract line</span></span>

<span data-ttu-id="f55d1-104">_**Aplica-se a:** Implementação leve - oportunidade potencial para fatura pró-forma_</span><span class="sxs-lookup"><span data-stu-id="f55d1-104">_**Applies To:** Lite deployment - deal to proforma invoicing_</span></span>

<span data-ttu-id="f55d1-105">Um item de contrato baseado em projeto tem componentes *incluídos* e componentes *faturáveis*.</span><span class="sxs-lookup"><span data-stu-id="f55d1-105">A project-based contract line has *included* components and *chargeable* components.</span></span>

<span data-ttu-id="f55d1-106">Os componentes incluídos são componentes que estão sujeitos a:</span><span class="sxs-lookup"><span data-stu-id="f55d1-106">Included components are components that are subject to:</span></span>

  - <span data-ttu-id="f55d1-107">Método de faturação e divisões de clientes</span><span class="sxs-lookup"><span data-stu-id="f55d1-107">Billing method and customer splits</span></span>
  - <span data-ttu-id="f55d1-108">Limites a não exceder</span><span class="sxs-lookup"><span data-stu-id="f55d1-108">Not-to-exceed limits</span></span> 
  - <span data-ttu-id="f55d1-109">Definições de frequência de fatura definidas no item de contrato baseado em projeto</span><span class="sxs-lookup"><span data-stu-id="f55d1-109">Invoice frequency settings defined on the project-based contract line</span></span>

<span data-ttu-id="f55d1-110">Um subconjunto dos componentes incluídos pode ser marcado como faturável utilizando o campo **Tipo de Faturação**.</span><span class="sxs-lookup"><span data-stu-id="f55d1-110">A subset of the included components can be marked as chargeable using the **Billing Type** field.</span></span> <span data-ttu-id="f55d1-111">O campo **Tipo de Faturação** é um conjunto de opções que permite os seguintes valores no contexto de um item de contrato:</span><span class="sxs-lookup"><span data-stu-id="f55d1-111">The **Billing Type** field is an option-set that allows the following values in the context of a contract line:</span></span>

  - <span data-ttu-id="f55d1-112">Faturável</span><span class="sxs-lookup"><span data-stu-id="f55d1-112">Chargeable</span></span>
  - <span data-ttu-id="f55d1-113">Não faturável</span><span class="sxs-lookup"><span data-stu-id="f55d1-113">Non-chargeable</span></span>

<span data-ttu-id="f55d1-114">Os componentes faturáveis podem ser definidos nas categorias de tarefas, funções e transações.</span><span class="sxs-lookup"><span data-stu-id="f55d1-114">Chargeable components can be defined on tasks, roles, and transaction categories.</span></span>

<span data-ttu-id="f55d1-115">A capacidade de faturação é definida em tarefas para um item de contrato de projeto e aplica-se a todas as classes de transação incluídas no item.</span><span class="sxs-lookup"><span data-stu-id="f55d1-115">Chargeability is defined on tasks for a project contract line and applies to all transaction classes included on the line.</span></span> <span data-ttu-id="f55d1-116">Se o campo **Incluir Tarefas** num item de contrato estiver em branco ou definido para \* *Todo o projeto* \*, o separador **Tarefas faturáveis** não estará disponível.</span><span class="sxs-lookup"><span data-stu-id="f55d1-116">If the **Include Tasks** field on a contract line is blank or set to \* *Entire project* , the **Chargeable tasks** tab will not be available.</span></span>

<span data-ttu-id="f55d1-117">A capacidade de faturação definida nas funções de um item de contrato de projeto aplica-se apenas à classe de transação **Tempo**.</span><span class="sxs-lookup"><span data-stu-id="f55d1-117">Chargeability defined on roles for a project contract line only applies to the **Time** transaction class.</span></span> <span data-ttu-id="f55d1-118">Se o campo **Incluir Tempo** num item de contrato estiver definido para **Não** , o separador **Funções faturáveis** não estará disponível.</span><span class="sxs-lookup"><span data-stu-id="f55d1-118">If the **Include Time** field on a contract line is set to **No** , the **Chargeable roles** tab will not be available.</span></span>

<span data-ttu-id="f55d1-119">A capacidade de faturação definida nas categorias de transação de um item de contrato de projeto aplica-se apenas à classe de transação **Despesa**.</span><span class="sxs-lookup"><span data-stu-id="f55d1-119">Chargeability defined on transaction categories for a project contract line only applies to the **Expense** transaction class.</span></span> <span data-ttu-id="f55d1-120">Se o campo **Incluir Despesas** estiver definido para **Não** , o separador **Categorias faturáveis** não estará disponível.</span><span class="sxs-lookup"><span data-stu-id="f55d1-120">If the **Include Expenses** field is set to **No** , the **Chargeable Categories** tab will not be available.</span></span>

### <a name="update-a-project-task-as-chargeable-or-non-chargeable"></a><span data-ttu-id="f55d1-121">Atualizar uma tarefa de projeto como faturável ou não faturável</span><span class="sxs-lookup"><span data-stu-id="f55d1-121">Update a project task as chargeable or non-chargeable</span></span>

<span data-ttu-id="f55d1-122">Uma tarefa do projeto pode ser faturável ou não faturável num item de contrato específico que torna possível a seguinte configuração:</span><span class="sxs-lookup"><span data-stu-id="f55d1-122">A project task can be chargeable or non-chargeable on a specific contract line which makes the following setup possible:</span></span>

<span data-ttu-id="f55d1-126">O tipo de faturação de uma tarefa pode ser configurado no separador **Tarefas Faturáveis** do item de contrato, atualizando o campo **Tipo de Faturação** na subgrelha de tarefas do item de contrato.</span><span class="sxs-lookup"><span data-stu-id="f55d1-126">A task's billing type can be configured on the **Chargeable Tasks** tab of the contract line by updating the **Billing Type** field on the contract line tasks sub-grid.</span></span> <span data-ttu-id="f55d1-127">Em alternativa, pode atualizar o campo **Tipo de Faturação** na subgrelha da configuração de Faturação de um projeto que mostra os itens de contratos associados a uma tarefa.</span><span class="sxs-lookup"><span data-stu-id="f55d1-127">Alternatively, you can update the **Billing Type** field on the sub-grid of the task Billing setup of a project that shows the contract lines associated to a task.</span></span>
