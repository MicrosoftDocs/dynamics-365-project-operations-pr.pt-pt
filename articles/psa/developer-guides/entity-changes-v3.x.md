<span data-ttu-id="66ff8-107">Nas versões do Dynamics 365 Project Service Automation (PSA) lançadas antes da versão 3.0, algumas das relações entre documentos de vendas, linhas de documentos de vendas e entidades de detalhe de linha de documento de vendas foram implementadas por meio de campos de cadeia que continham uma representação em cadeia do GUID da entidade relacionada.</span><span class="sxs-lookup"><span data-stu-id="66ff8-107">In versions of Dynamics 365 Project Service Automation (PSA) released prior to version 3.0, some of the relationships between sales documents, sales document lines, and sales document line detail entities were implemented through string fields that would hold a string representation of GUID of the related entity.</span></span> <span data-ttu-id="66ff8-108">Isso ocorreu devido às limitações da plataforma que exigiam um código personalizado significativo nos lados do servidor e do cliente da solução para fazer com que essas relações funcionassem de maneira semelhante às relações de entidade típicas do Dynamics CRM e para fazer com que os campos de cadeia agissem como campos de pesquisa.</span><span class="sxs-lookup"><span data-stu-id="66ff8-108">This was due to platform limitations that required significant custom code on the server and client sides of the solution to make those relationships work similar to typical Dynamics CRM entity relationships and to make string fields act like lookup fields.</span></span>

<span data-ttu-id="66ff8-110">Como os campos de pesquisa agora podem ser utilizados para indicar referências a entidades, os campos que mantinham o valor da cadeia do GUID da entidade relacionada nas versões anteriores já não são necessários e, portanto, foram preteridos.</span><span class="sxs-lookup"><span data-stu-id="66ff8-110">Because lookup fields can now be used to indicate references to entities, the fields that held the string value of the GUID of the related entity in previous versions are no longer needed and therefore have been deprecated.</span></span> <span data-ttu-id="66ff8-111">O código personalizado do lado do cliente e do servidor que lida com as relações definidas pelos campos de cadeia legados também foi preterido.</span><span class="sxs-lookup"><span data-stu-id="66ff8-111">The custom client and server side code that handles the relationships defined by legacy string fields has also been deprecated.</span></span>
