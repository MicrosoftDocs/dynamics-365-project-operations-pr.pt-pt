# <a name="set-up-and-use-pay-when-paid-vendor-payments"></a><span data-ttu-id="18b1f-103">Configurar e utilizar pagamentos de fornecedor paga quando pago</span><span class="sxs-lookup"><span data-stu-id="18b1f-103">Set up and use pay-when-paid vendor payments</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="18b1f-104">Quando aprova um fornecedor para trabalhar como subcontratado, poderá pretender reter o pagamento ao fornecedor até que o seu cliente lhe pague pelo projeto.</span><span class="sxs-lookup"><span data-stu-id="18b1f-104">When you approve a vendor to work as a subcontractor, you might want to withhold payment to the vendor until your customer pays you for the project.</span></span> <span data-ttu-id="18b1f-105">Para suportar este cenário, pode configurar os termos de paga quando pago (PWP) quando configurar a nota de encomenda (PO) com o fornecedor.</span><span class="sxs-lookup"><span data-stu-id="18b1f-105">To support this scenario, you can set up pay-when-paid (PWP) terms when you set up the purchase order (PO) with the vendor.</span></span>

<span data-ttu-id="18b1f-106">Por exemplo, quando um cliente paga um valor numa fatura do projeto, pode libertar parte ou o total do montante da fatura do fornecedor.</span><span class="sxs-lookup"><span data-stu-id="18b1f-106">For example, when a customer pays an amount on a project invoice, you can release some or all of the vendor invoice amount.</span></span> <span data-ttu-id="18b1f-107">Basta configurar os termos PWP que especificam se o fornecedor será pago depois de receber uma percentagem do pagamento relacionado do cliente.</span><span class="sxs-lookup"><span data-stu-id="18b1f-107">Just set up PWP terms that specify that the vendor will be paid after you receive a percentage of the related payment from the customer.</span></span> <span data-ttu-id="18b1f-108">Se receber o pagamento parcial de um cliente, pode libertar manualmente algumas das faturas relacionadas do fornecedor para pagamento.</span><span class="sxs-lookup"><span data-stu-id="18b1f-108">If you receive partial payment from a customer, you can manually release some of the related vendor invoices for payment.</span></span>

<span data-ttu-id="18b1f-109">O exemplo a seguir descreve o processo quando são utilizados os termos PWP.</span><span class="sxs-lookup"><span data-stu-id="18b1f-109">The following example outlines the process when PWP terms are used.</span></span>

## <a name="example"></a><span data-ttu-id="18b1f-110">Exemplo</span><span class="sxs-lookup"><span data-stu-id="18b1f-110">Example</span></span>

<span data-ttu-id="18b1f-111">A sua organização concorda em fornecer a um cliente 100 computadores que tenham software instalado, por um preço de 150,00 dólares americanos (USD) por computador.</span><span class="sxs-lookup"><span data-stu-id="18b1f-111">Your organization agrees to provide a customer with 100 computers that have software installed, for a price of 150.00 US dollars (USD) per computer.</span></span> <span data-ttu-id="18b1f-112">Em seguida, contrata um fornecedor para lhe fornecer os computadores que têm software instalado.</span><span class="sxs-lookup"><span data-stu-id="18b1f-112">You then hire a vendor to provide you with the computers that have software installed.</span></span> <span data-ttu-id="18b1f-113">De acordo com o contrato, o cliente deve aprovar a qualidade dos computadores antes de pagar à sua organização.</span><span class="sxs-lookup"><span data-stu-id="18b1f-113">According to the agreement, the customer must approve the quality of the computers before it pays your organization.</span></span> <span data-ttu-id="18b1f-114">A política da sua organização é reter o pagamento aos fornecedores até que o cliente lhe pague.</span><span class="sxs-lookup"><span data-stu-id="18b1f-114">Your organization's policy is to withhold payment to vendors until the customer has paid you.</span></span> <span data-ttu-id="18b1f-115">Por isso, configura o projeto para que tenha uma percentagem PWP de 100 por cento.</span><span class="sxs-lookup"><span data-stu-id="18b1f-115">Therefore, you set up the project so that it has a PWP percentage of 100 percent.</span></span>

<span data-ttu-id="18b1f-116">O fornecedor envia-lhe os 100 computadores que têm software instalado, juntamente com uma fatura para 10.000,00 USD.</span><span class="sxs-lookup"><span data-stu-id="18b1f-116">The vendor sends you the 100 computers that have software installed, together with an invoice for 10,000.00 USD.</span></span> <span data-ttu-id="18b1f-117">Como os termos PWP estão configurados para o seu projeto, não paga ao fornecedor após a receção dos computadores.</span><span class="sxs-lookup"><span data-stu-id="18b1f-117">Because PWP terms are set up for your project, you don't pay the vendor upon receipt of the computers.</span></span> <span data-ttu-id="18b1f-118">Em vez disso, envia os computadores para o cliente, juntamente com uma fatura para 15.000,00.</span><span class="sxs-lookup"><span data-stu-id="18b1f-118">Instead, you send the computers to the customer, together with an invoice for 15,000.00.</span></span> <span data-ttu-id="18b1f-119">O cliente inspeciona os computadores e aprova a totalidade da fatura do projeto.</span><span class="sxs-lookup"><span data-stu-id="18b1f-119">The customer inspects the computers and approves the full amount of the project invoice.</span></span>

<span data-ttu-id="18b1f-120">Depois de receber o pagamento total do cliente, paga ao vendedor 10.000,00, o valor total da fatura do fornecedor.</span><span class="sxs-lookup"><span data-stu-id="18b1f-120">After you receive the full payment from the customer, you pay the vendor 10,000.00, the full amount of the vendor invoice.</span></span>

<span data-ttu-id="18b1f-122">Ao configurar os termos PWP para um projeto, deve especificar, em percentagem, o montante mínimo que um cliente deve pagar-lhe pelo projeto antes de pagar ao fornecedor.</span><span class="sxs-lookup"><span data-stu-id="18b1f-122">When you set up PWP terms for a project, you must specify, as a percentage, the minimum amount that a customer must pay you for the project before you will pay the vendor.</span></span> <span data-ttu-id="18b1f-123">O limiar é calculado automaticamente nas faturas do cliente para o projeto.</span><span class="sxs-lookup"><span data-stu-id="18b1f-123">The threshold amount is automatically calculated on the customer invoices for the project.</span></span> <span data-ttu-id="18b1f-124">Siga estes passos para configurar a percentagem de limiar para os termos PWP.</span><span class="sxs-lookup"><span data-stu-id="18b1f-124">Follow these steps to set up the threshold percentage for PWP terms.</span></span>

3. <span data-ttu-id="18b1f-128">No campo **Código da conta**, selecione uma das seguintes opções:</span><span class="sxs-lookup"><span data-stu-id="18b1f-128">In the **Account code** field, select one of the following options:</span></span>

    - <span data-ttu-id="18b1f-129">**Tabela** – Os termos PWP aplicam-se a um único fornecedor.</span><span class="sxs-lookup"><span data-stu-id="18b1f-129">**Table** – The PWP terms apply to a single vendor.</span></span>
    - <span data-ttu-id="18b1f-130">**Grupo** – Os termos PWP aplicam-se a todos os fornecedores de um grupo de fornecedores.</span><span class="sxs-lookup"><span data-stu-id="18b1f-130">**Group** – The PWP terms apply to all vendors in a vendor group.</span></span>
    - <span data-ttu-id="18b1f-131">**Todos** – Os termos PWP aplicam-se a todos os fornecedores.</span><span class="sxs-lookup"><span data-stu-id="18b1f-131">**All** – The PWP terms apply to all vendors.</span></span>

4. <span data-ttu-id="18b1f-132">Se selecionou **Tabela** ou **Grupo** no passo anterior, no campo **Fornecedor/Grupo de Fornecedores**, selecione o fornecedor ou grupo de fornecedores a que os termos PWP se aplicam.</span><span class="sxs-lookup"><span data-stu-id="18b1f-132">If you selected **Table** or **Group** in the previous step, in the **Vendor/Vendor group** field, select the vendor or vendor group that the PWP terms apply to.</span></span> <span data-ttu-id="18b1f-133">Se selecionou **Todos** no passo anterior, o campo **Fornecedor/Grupo de Fornecedores** não pode ser editado.</span><span class="sxs-lookup"><span data-stu-id="18b1f-133">If you selected **All** in the previous step, the **Vendor/Vendor group** field can't be edited.</span></span>
5. <span data-ttu-id="18b1f-134">Se forem configurados termos de sinal do fornecedor para o fornecedor no projeto, no campo **Termos de sinal do fornecedor**, selecione o ID da regra para os termos de sinal.</span><span class="sxs-lookup"><span data-stu-id="18b1f-134">If vendor retention terms are set up for the vendor in the project, in the **Vendor retention terms** field, select the rule ID for the retention terms.</span></span>
