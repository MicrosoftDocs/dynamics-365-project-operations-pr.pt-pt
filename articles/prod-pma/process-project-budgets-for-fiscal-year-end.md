## <a name="review-and-analyze-remaining-budget-amounts"></a><span data-ttu-id="c62f2-107">Rever e analisar os montantes orçamentais remanescentes</span><span class="sxs-lookup"><span data-stu-id="c62f2-107">Review and analyze remaining budget amounts</span></span>

2. <span data-ttu-id="c62f2-110">Na página **Processo de transposição do orçamento do projeto** no separador **Opções de final de ano**, verifique se **Montantes orçamentais de projeto restantes transpostos** não estão ativados.</span><span class="sxs-lookup"><span data-stu-id="c62f2-110">On the **Project budget carry-forward process** page, on the **Year-end options** tab, verify that **Carry forward remaining project budget amounts** is not enabled.</span></span>
3. <span data-ttu-id="c62f2-111">No separador **Parâmetros**, no campo **Ano de orçamento do projeto**, selecione o ano fiscal para o qual pretende ver o restante montante orçamental.</span><span class="sxs-lookup"><span data-stu-id="c62f2-111">On the **Parameters** tab, in the **Project budget year** field, select the fiscal year for which you want to view the remaining budget amount.</span></span>

6. <span data-ttu-id="c62f2-114">Para incluir projetos que satisfaçam os seus critérios selecionados e não tenham orçamento remanescente, selecione **Mostrar zero restantes**.</span><span class="sxs-lookup"><span data-stu-id="c62f2-114">To include projects that meet your selected criteria and have no remaining budget, select **Show zero remaining**.</span></span>  
7. <span data-ttu-id="c62f2-115">No separador **Selecionar Orçamentos**, selecione **Obter todos os orçamentos** para carregar todos os orçamentos que correspondam aos critérios selecionados e, em seguida, selecione **Processar**.</span><span class="sxs-lookup"><span data-stu-id="c62f2-115">On the **Select Budgets** tab, select **Retrieve all budgets** to load all budgets that match your selected criteria, and then select **Process**.</span></span>

<span data-ttu-id="c62f2-117">Para obter mais informações sobre uma linha específica no painel, selecione a linha e, em seguida, selecione **Ver detalhes do orçamento** ou **Ver contas**.</span><span class="sxs-lookup"><span data-stu-id="c62f2-117">For more information about a specific line in the pane, select the line, and then select **View budget details** or **View accounts**.</span></span>
