## <a name="labor-cost-tracking-view"></a><span data-ttu-id="eee09-108">Vista de monitorização de custo de mão de obra</span><span class="sxs-lookup"><span data-stu-id="eee09-108">Labor Cost Tracking view</span></span>

<span data-ttu-id="eee09-109">Na página **Projetos**, no separador **Monitorização**, pode selecionar **Monitorização** > **Custo** para abrir a vista de **Monitorização de custo** e ver o progresso do gasto de mão de obra em cada tarefa no plano do projeto.</span><span class="sxs-lookup"><span data-stu-id="eee09-109">On the **Projects** page, on the **Tracking** tab, you can select **Tracking** > **Cost** to open the **Cost Tracking** view and see the progress of labor spend on each task in the project plan.</span></span> <span data-ttu-id="eee09-110">Esta perspetiva também compara o custo real da mão de obra gasto numa tarefa com o custo de mão de obra planeada da tarefa.</span><span class="sxs-lookup"><span data-stu-id="eee09-110">This view also compares the actual labor cost spent on a task to the task's planned labor cost.</span></span> <span data-ttu-id="eee09-111">O Project Operations utiliza as seguintes fórmulas para calcular as métricas do custo da mão de obra:</span><span class="sxs-lookup"><span data-stu-id="eee09-111">Project Operations uses the following formulas to calculate labor cost metrics:</span></span>

- <span data-ttu-id="eee09-112">**Custo previsto**: Custo estimado de todas as atribuições de recursos em cada tarefa do nó folha</span><span class="sxs-lookup"><span data-stu-id="eee09-112">**Planned cost**: Estimated cost of all resource assignments on each leaf node task</span></span>

- <span data-ttu-id="eee09-116">**Custo na conclusão**: Custo restante – Custo real</span><span class="sxs-lookup"><span data-stu-id="eee09-116">**Cost at Complete**: Remaining cost + Actual cost</span></span>
- <span data-ttu-id="eee09-117">**Variação de custo**: Custo planificado – Estimativa de custo na conclusão</span><span class="sxs-lookup"><span data-stu-id="eee09-117">**Cost variance**: Planned cost – Cost estimate at complete</span></span>

<span data-ttu-id="eee09-118">Cada tarefa apresenta uma projeção do desvio de custo na tarefa.</span><span class="sxs-lookup"><span data-stu-id="eee09-118">Each task shows a projection of the cost variance on the task.</span></span> <span data-ttu-id="eee09-119">Se a estimativa de custos na conclusão for superior ao custo previsto, prevê-se que a tarefa ultrapasse o orçamento.</span><span class="sxs-lookup"><span data-stu-id="eee09-119">If the cost estimate at complete is more than the planned cost, the task is projected to go over budget.</span></span> <span data-ttu-id="eee09-120">Se a estimativa de custos na conclusão for inferior ao custo previsto, prevê-se que a tarefa termine abaixo do orçamento.</span><span class="sxs-lookup"><span data-stu-id="eee09-120">If the cost estimate at complete is less than the planned cost, the task is projected to finish under budget.</span></span>

>[!NOTE]
> <span data-ttu-id="eee09-121">O Project Operations só mostra os custos de mão de obra na página do **Projeto** no separador **Monitorização**. <span data-ttu-id="eee09-122">Todos os montantes de custos apresentados são convertidos para a moeda de custo do projeto a partir da moeda de preço do projeto utilizada para determinar a taxa de custo.</span><span class="sxs-lookup"><span data-stu-id="eee09-122">All cost amounts shown are converted to the project's cost currency from the project's cost price currency used to determine the cost rate.</span></span> <span data-ttu-id="eee09-123">A moeda de custo do projeto é a moeda da unidade de contratação do projeto.</span><span class="sxs-lookup"><span data-stu-id="eee09-123">The cost currency of the project is the currency of the contracting unit on the project.</span></span> <span data-ttu-id="eee09-124">Os valores de custo estimados para o tempo que são mostrados no separador **Estimativas** na página do **Projeto** podem não somar o custo planeado no separador **Monitorização**. >
> - <span data-ttu-id="eee09-125">O separador **Estimativa** calcula o custo estimado utilizando a mesma moeda da taxa de custo na tabela de preços.</span><span class="sxs-lookup"><span data-stu-id="eee09-125">The **Estimates tab** calculates the estimated cost by using the same currency of the cost rate in the price list.</span></span> <span data-ttu-id="eee09-126">Em seguida, o custo estimado na moeda da lista de preços é convertido para o custo estimado na moeda de custo do projeto.</span><span class="sxs-lookup"><span data-stu-id="eee09-126">Then, the estimated cost in the price list currency is converted to the estimated cost in the project's cost currency.</span></span> <span data-ttu-id="eee09-127">O custo estimado na moeda do projeto é apresentado arredondado para 2 casas decimais.</span><span class="sxs-lookup"><span data-stu-id="eee09-127">The estimated cost in the project currency is shown rounded to 2 decimal places.</span></span> <span data-ttu-id="eee09-128">Em nenhum momento durante esta conversão é aplicada a precisão cambial.</span><span class="sxs-lookup"><span data-stu-id="eee09-128">At no point during this conversion is currency precision applied.</span></span>

   ><li><span data-ttu-id="eee09-130">O valor estimado do custo na moeda da lista de preços é convertido para a moeda base (conversão 1).</span><span class="sxs-lookup"><span data-stu-id="eee09-130">The estimated cost amount in the price list currency is converted to the base currency (conversion 1).</span></span></li>
   ><li><span data-ttu-id="eee09-131">O valor estimado do custo na moeda base é convertido para a moeda do projeto (conversão 2).</span><span class="sxs-lookup"><span data-stu-id="eee09-131">The estimated cost amount in the base currency is converted to the project cost currency (conversion 2).</span></span> </li>
   ></ol>
   ><span data-ttu-id="eee09-132">A precisão da moeda é aplicada em ambas as etapas para obter um custo planeado (no separador **Monitorização**) que se desvia ligeiramente do custo estimado (na vista **tempo - vista faseada** no separador **Estimativas**).</span><span class="sxs-lookup"><span data-stu-id="eee09-132">Currency precision is applied in both steps to get a planned cost (on the **Tracking** tab) that deviates slightly from the estimated cost (on the **Time - phased** view on the **Estimates** tab).</span></span>
