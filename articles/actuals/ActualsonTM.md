# <a name="actuals-impact-in-a-time-and-materials-engagement"></a>Impacto do valor real numa cativação de tempo e materiais

_**Aplica-se a:** Project Operations para cenários baseados em recursos/não armazenados, implementação leve - negócio para faturação pró-forma_

A tabela seguinte lista os valores reais de diferentes tipos de transações criados em vários eventos numa cativação de tempo e materiais.

| Event | Custo real | Valor real de vendas não faturadas | Valor real de vendas faturadas | Exemplo |
|---|---|---|---|---|
| O tempo foi criado. | Não aplicável | Não aplicável | Não aplicável | <p>Roberto Neves, da unidade organizacional Fabrikam US, que tem uma taxa de custo de 100 USD por hora, está a trabalhar num projeto denominado "Instalação Neves na Adatum". Neste projeto, a taxa de faturação contratual é 200 USD por hora. Eis uma amostra de entrada de tempo Roberto Neves:</p><p>Bob Kozack, 8 horas</p> |
| O tempo foi submetido. | Não aplicável | Não aplicável | Não aplicável | Para a entrada de tempo, são criadas uma linha do diário de custo e diários de vendas não faturadas. O preço predefinido e a taxa de custo são inseridos na entrada de diário. |
| A entrada de tempo é revocada antes de ser aprovada. | Não aplicável | Não aplicável | Não aplicável | |
| O tempo é aprovado: horas submetidas equivalem a horas a faturar. | É criado um custo real. | É criado um valor real de vendas não faturadas. | Não aplicável | <p>São criados novos valores reais:</p><ul><li>**Custo real:** Roberto Neves, 8 horas, 800 USD</li><li>**Valor real de vendas não faturadas**: Roberto Neves, 8 horas, 1.600 USD</li></ul> |
| O tempo é aprovado: as horas a faturar são reduzidas abaixo das horas submetidas. | É criado um custo real. | <p>São criados dois valores reais de vendas não faturadas:</p><ul><li>valor real de vendas não faturadas cobráveis para as horas a faturar</li><li>valor real de vendas não faturadas não-cobráveis para a restante quantidade</li></ul> | Não aplicável | <p>São criados novos valores reais:</p><ul><li>**Custo real:** Roberto Neves, 8 horas, 800 USD</li><li>**Valor real de vendas não faturadas:** Bob Kozack, 6 horas, 1.200 USD, *Cobráveis*</li><li>**Valor real de vendas não faturadas:** Bob Kozack, 2 horas, 400 USD, *Não cobráveis*</li></ul> |
| O tempo é aprovado: as horas a faturar são aumentadas acima das horas submetidas. | É criado um custo real. | É criado um valor real de vendas não faturadas. | Não aplicável | <p>São criados novos valores reais:</p><ul><li>**Custo real:** Roberto Neves, 8 horas, 800 USD</li><li>**Valor real de vendas não faturadas**: Bob Kozack, 10 horas, 2.000 USD</li></ul> |
| A aprovação do tempo foi cancelada. | <p>O estado de ajuste do valor real do custo original é atualizado para **Ajustado**.</p><p>É criado um valor real de custo que tem um estado de ajuste de **Não ajustável**.</p> | <p>O estado de ajuste do valor real das vendas não faturadas original é atualizado para **Ajustado**.</p><p>É criado um valor real de vendas não faturadas que tem um estado de ajuste **Não ajustável**.</p> | Não aplicável | <p>Valores reais existentes que são atualizados:</p><ul><li>**Custo real:** Roberto Neves, 8 horas, 800 USD, *Ajustado*</li><li>**Valor real de vendas não faturadas:** Bob Kozack, 8 horas, 1.600 USD, *Ajustado*</li></ul><p>Novos valores reais criados para reverter o impacto financeiro anterior:</p><ul><li>**Custo real:** Roberto Neves, (8 horas), (800 USD), *Não ajustado*</li><li>**Valor real de vendas não faturadas:** Bob Kozack, (8 horas), (1.600 USD), *Não ajustável*</li></ul> |
| A entrada de tempo é revocada depois de ser aprovada. | <p>O estado de ajuste do valor real do custo original é atualizado para **Ajustado**.</p><p>É criado um valor real de custo que tem um estado de ajuste de **Não ajustável**.</p> | <p>O estado de ajuste do valor real das vendas não faturadas original é atualizado para **Ajustado**.</p><p>É criado um valor real de vendas não faturadas que tem um estado de ajuste **Não ajustável**.</p> | Não aplicável | <p>Valores reais existentes que são atualizados:</p><ul><li>**Custo real:** Roberto Neves, 8 horas, 800 USD, *Ajustado*</li><li>**Valor real de vendas não faturadas:** Bob Kozack, 8 horas, 1.600 USD, *Ajustado*</li></ul><p>Novos valores reais criados para reverter o impacto financeiro anterior:</p><ul><li>**Custo real:** Roberto Neves, (8 horas), (800 USD), *Não ajustado*</li><li>**Valor real de vendas não faturadas:** Bob Kozack, (8 horas), (1.600 USD), *Não ajustável*</li></ul> |
| O contrato está confirmado. | <p>O estado de ajuste dos valores reais de custo antigo é atualizado para **Ajustado**.</p><p>São criados valores reais de custo de reversão que têm um estado de ajuste **Não ajustável**.</p><p>Os novos valores reais de custo são criados após a reavaliação das regras contratuais.</p> | <p>O estado de ajuste do valor real das vendas não faturadas antigo é atualizado para **Ajustado**.</p><p>São criados valores reais de vendas não faturadas de reversão que têm um estado de ajuste **Não ajustável**.</p><p>Os novos valores reais de vendas não faturados são criados após a reavaliação das regras contratuais.</p> | Não aplicável | <p>Valores reais existentes que são atualizados:</p><ul><li>**Custo real:** Roberto Neves, 8 horas, 800 USD, *Ajustado*</li><li>**Valor real de vendas não faturadas:** Bob Kozack, 8 horas, 1.600 USD, *Ajustado*</li></ul><p>Novos valores reais criados para reverter o impacto financeiro anterior:</p><ul><li>**Custo real:** Roberto Neves, (8 horas), (800 USD), *Não ajustado*</li><li>**Valor real de vendas não faturadas:** Bob Kozack, (8 horas), (1.600 USD), *Não ajustável*</li></ul><p>Os novos valores reais que são criados para reavaliar o impacto financeiro anterior:</p><ul><li>**Custo real:** Roberto Neves, 8 horas, 800 USD</li><li>**Valor real de vendas não faturadas**: Roberto Neves, 8 horas, 1.600 USD</li></ul> |
| É criada uma fatura. | Não aplicável | Não aplicável | Não aplicável | |
| A fatura é confirmada. Não existe nenhuma alteração na quantidade no detalhe da linha de fatura da quantidade no valor real de vendas não faturadas. | Não aplicável | <p>O estado da fatura do valor real de vendas não faturadas antigo é atualizado.</p><p>São criados valores reais de vendas não faturadas de reversão que têm um estado de ajuste **Não ajustável**. | É criado um valor real de vendas faturadas. | <p>Valor real existente que se mantém inalterado:</p><ul><li>**Custo real:** Roberto Neves, 8 horas, 800 USD</li></ul><p>Valor real existentes que foi atualizado:</p><ul><li>**Valor real de vendas não faturadas:** Bob Kozack, 8 horas, 1.600 USD, *Fatura do Cliente Publicada*</li></ul>O novo valor real que é criado para reverter o trabalho financeiro em curso (WIP):</p><ul><li>**Valor real de vendas não faturadas**: Bob Kozack, (8 horas), (1.600 USD)</li></ul><p>O novo valor real que é criado para gravar os valores de vendas faturados:</p><ul><li>**Valor real de vendas faturadas**: Bob Kozack, 8 horas, 1.600 USD</li></ul> |
| A fatura é confirmada depois de a quantidade no detalhe no da linha de fatura ser subtraído da quantidade no valor real de vendas não faturadas. | Não aplicável | <p>O estado de ajuste dos valores reais de vendas não faturadas originais é atualizado para **Ajustado**.</p><p>Os valores reais de vendas não faturadas de reversão são criados para os valores reais de vendas não faturadas originais. Têm um estado de ajuste **Não ajustável**.</p><p>São criados dois novo valores reais de vendas não faturadas:</p><ul><li>valor real de vendas não faturadas cobráveis para as horas a faturar</li><li>valor real de vendas não faturadas não-cobráveis para a restante quantidade</li></ul><p>Os valores reais de vendas não faturadas de reversão são criados para os dois novos valores reais de vendas não faturadas.</p> | <p>São criados dois valores reais de vendas faturadas:</p><ul><li>Valor real de vendas faturadas cobráveis para as horas a faturar</li><li>Valor real de vendas faturadas não-cobráveis para a restante quantidade</li></ul> | <p>Valor real existente que se mantém inalterado:</p><ul><li>**Custo real:** Roberto Neves, 8 horas, 800 USD</li></ul><p>Valor real existentes que foi atualizado:</p><ul><li>**Valor real de vendas não faturadas:** Bob Kozack, 8 horas, 1.600 USD, *Ajustado*</li></ul><p>O novo valor real que é criados para reverter o impacto financeiro anterior WIP:</p><ul><li>**Valor real de vendas não faturadas:** Bob Kozack, (8 horas), (1.600 USD), *Não ajustável*</li></ul><p>Os novos valores reais que são criados para gravar as vendas atualizadas WIP:</p><ul><li>**Valor real de vendas não faturadas:** Bob Kozack, 6 horas, 1.200 USD, *Cobráveis*</li><li>**Valor real de vendas não faturadas:** Bob Kozack, 2 horas, 400 USD, *Não cobráveis*</li></ul><p>Os novos valores reais que são criados para reverter as vendas atualizadas WIP:</p><ul><li>**Valor real de vendas não faturadas:** Bob Kozack, (6 horas), (1.200 USD), *Cobráveis*</li><li>**Valor real de vendas não faturadas:** Bob Kozack, (2 horas), (400 USD), *Não cobráveis*</li></ul><p>Os novos valores reais que são criados para gravar os valores de vendas faturados:</p><ul><li>**Valor real de vendas faturadas:** Bob Kozack, 6 horas, 1.200 USD, *Cobráveis*</li><li>**Valor real de vendas faturadas:** Bob Kozack, 2 horas, 400 USD, *Não cobráveis*</li></ul> |
| A fatura é confirmada depois de a quantidade no detalhe no da linha de fatura ser reduzido da quantidade no valor real de vendas não faturadas. | Não aplicável | <p>O estado de ajuste dos valores reais de vendas não faturadas originais é atualizado para **Ajustado**.</p><p>Os valores reais de vendas não faturadas de reversão são criados para os valores reais de vendas não faturadas originais. | A fatura é corrigida para diminuir a quantidade faturável ou o preço. | Não aplicável | <p>São criados dois valores reais de vendas não faturadas:</p><ul><li>Valor real de vendas não faturadas cobráveis para a quantidade na fatura corretiva</li><li>Valor real de vendas não faturadas cobráveis para a restante quantidade</li></ul><p>Os valores reais de vendas não faturadas de reversão são criados para os dois novos valores reais de vendas não faturadas.</p> | <p>São criados valores reais de vendas faturadas de reversão:</p><p>São criados novos valores reais de vendas faturadas para a nova quantidade. | <p>Valores reais existentes que se mantêm inalterados:</p><ul><li>**Custo real:** Roberto Neves, 8 horas, 800 USD</li><li>**Valor real de vendas não faturadas:** Bob Kozack, 8 horas, 1.600 USD, *Fatura do Cliente Publicada*</li><li>**Valor real de vendas não faturadas**: Bob Kozack, (8 horas), (1.600 USD)</li></ul><p>Valor real existentes que foi atualizado:</p><ul><li>**Valor real de vendas faturadas:** Bob Kozack, (8 horas), (1.600 USD), *Ajustado*</li></ul><p>O novo valor real que é criado para reverter os valores de vendas faturados anteriores:</p><ul><li>**Valor real de vendas faturadas:** Bob Kozack, (8 horas), (1.600 USD), *Não ajustável*</li></ul><p>Os novos valores reais que são criados para gravar as vendas corrigidas WIP:</p><ul><li>**Valor real de vendas não faturadas**: Bob Kozack, 6 horas, 1.200 USD, *Cobrável*, *Fatura do cliente publicada*</li><li>**Valor real de vendas não faturadas:** Bob Kozack, 2 horas, 400 USD, *Cobráveis*</li></ul><p>O novo valor real que é criado para reverter as vendas corrigidas WIP:</p><ul><li>**Valor real de vendas não faturadas:** Bob Kozack, (6 horas), (1.200 USD), *Cobráveis*, *Não ajustável*</li></ul><p>O novo valor real que é criado para gravar os valores de vendas faturados corrigidos:</p><ul><li>**Valor real de vendas faturadas:** Bob Kozack, 6 horas, 1.200 USD, *Cobráveis*</li></ul> |
